PlotCDF
^^^^^^^^^^^^
This python package allows us to plot beautiful cumulative distributions functions and quantile functions
for discrete random variables.

Installation
^^^^^^^^^^^^^^
```
$ pip install plotcdf
```

Usage
^^^^^^^^^^^^^^
Get it in the [GitHub](https://github.com/parcr/plotcdf)
